# Examples

This folder contains walkthrough examples showing how the agentic enterprise operating model works end-to-end.

## Available Examples

| Example | Description |
|---------|-------------|
| [Generic Feature Lifecycle](generic-feature-lifecycle.md) | End-to-end lifecycle of a feature from signal through deployment |

## How to Use Examples

These examples serve as:
1. **Onboarding material** — New team members can trace a feature through the full lifecycle
2. **Reference implementation** — See how signals, missions, templates, and loops connect
3. **Customization guide** — Understand what needs to change for your organization

When customizing the framework, create your own examples in this folder that reflect your company's specific ventures, divisions, and workflows.
