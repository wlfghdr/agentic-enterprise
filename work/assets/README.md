## How to Register an Asset

1. Use the template: `work/assets/_TEMPLATE-asset-registry-entry.md`
2. Name the file: `<asset-name>.md`
3. Submit as a Pull Request

## Naming Convention

- `<descriptive-asset-name>.md` (lowercase, hyphen-separated)

## Asset Statuses

| Status | Meaning |
|--------|---------|
| **draft** | Asset being created |
| **review** | Under quality review |
| **published** | Live and available |
| **archived** | No longer current, kept for reference |
| **deprecated** | Superseded, migration path available |
